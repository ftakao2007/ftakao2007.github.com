Ec2-api-tools

Apr 15th, 2013

初期設定

  • AWSのEC2 API Toolsをインストールなしで利用する方法
    ➡基本的にこの設定でOK。 AWSのログインとかがあるメニューから証明書取得の画面を開き、証明書と秘密鍵を取得する。

  • 2013/03/31
    こちらのアクセスキーとシークレットキーでアクセスする方法に変更
    EC2 API ToolsでアクセスキーIDとシークレットアクセスキーを使う

諸情報

実行ファイルのPATH

/opt/aws/bin

環境変数

EC2_HOME

ec2-api-toolsのHOME * 例
export EC2_HOME=/opt/aws/apitools/ec2

EC2_PRIVATE_KEY

秘密鍵のPATH。今は使ってない

  • 例
    export EC2_PRIVATE_KEY=$EC2_HOME/pk-XXXXXXXXXXXXXXXXXXXXX.pem

EC2_CERT

証明書のPATH。今は使ってない。

  • 例
    export EC2_CERT=$EC2_HOME/cert-XXXXXXXXXXXXXXXXXXXXXXX.pem

AWS_ACCESS_KEY

AWSアクセスキー

  • 例
    export AWS_ACCESS_KEY=xxxxxxxxxxxxxxxxxx

AWS_SECRET_KEY

AWSシークレットキー

  • 例
    export AWS_SECRET_KEY=yyyyyyyyyyyyyyyyyyyyyyyyyyy

EC2_URL

リージョンの指定

  • 例
    export EC2_URL=http://ec2.ap-northeast-1.amazonaws.com

コマンド

確認

ec2ver

バージョンの表示

ec2-describe-images

AMIの表示

インスタンス操作関連

ec2-describe-instances

インスタンスの一覧の情報が表示される

※ 引数にインスタンスIDを指定数と、そのインスタンスの情報のみ表示される

ec2-start-instances

インスタンスを起動

ec2-stop-instances

インスタンスを停止

起動しているインスタンスのみ確認する場合は以下

snapshot関連

ec2-describe-snapshots

snapshot一覧の表示

ec2-copy-snapshot

snapshotのコピー

※東京からシンガポールのリージョンへコピー

ec2-describe-snapshot

snapshotのアクセス権限確認

権限が設定されていない場合(非公開設定)は何も出力されない

ec2-modify-snapshot-attribute

snapshotのアクセス権限を変更する

※すべてのユーザに公開される設定

ec2-reset-snapshot-attribute

権限をリセットする

volume関連

ec2-detach-volume

ボリュームをデタッチする

ec2-attach-volume

ボリュームをアタッチする

ec2-create-volume

ボリュームを作成する

Elastic IP関連

ec2-describe-addresses

トラブルシューティング

Instance does not have a volume attached at root (/dev/sda1)

/dev/sdfになっていたのを/dev/sda1でアタッチし直したらOK

Sed

Apr 13th, 2013

sed (ストリーミングエディタ)

Option

n

標準出力を制御

  • pで表示する場合はnをつけると目的の部分だけ出力できる
  • dで削除する場合はnを付けなければ削除された行以外を出力できる

e

次にくる値はコマンドとみなす

f

次にくる値はスクリプトとみなす

i

標準出力に出力せず、直接ファイルを書き換える

Command

p

出力する

d

削除する

q

特定の文字列が含まれる行まで表示

s

置換する

  • g すべてのパターンに対して処理を実行する
  • n(整数) n番目の発生を置換
  • p パターンスペースの先頭セグメントから最初の復帰改行までを標準出力に出力する
  • w ファイルへ書き出す

Match

空行以外を出力

空行を出力

最初の空行から次の空行までを出力

最初の空行から次の空行までの行を削除

Advance

正規表現(?)

&

置換のときに省略して記載できる

参考

  • sed - ストリーミングエディタ
  • sed に関するメモ
