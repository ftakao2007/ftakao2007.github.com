リモートリポジトリ

  • gitサーバ側に作成されるリポジトリ。クライアント側から作成することはできない。
  • このリポジトリが無いとpushできない

master

  • masterのブランチ
  • git commit -m "hoge" のように、commitを初めてした時に自動で作成される
  • ブランチは git branch で確認可能

origin

  • URLのエイリアスみたいなもの
  • git remote add origin https://github.com/ftakao2007/test.git で後ろのURLが登録される
