Octopress使い方

まとめに利用

  • EC2にRedmineを入れてそこにまとめているが、単純にまとめるだけならブログの方が情報が共有できる。

  • パスワードを入力する必要もなくアクセスが容易

  • Redmineのためだけに立ち上げているインスタンスを落とすことで経費が削減できる

やること

いずれできたらいいなぁくらいの温度感

Octopressについて

  • iPadで日本語の読み書き → nkfインストール

→ 後回し

  • 今のところローカルMacのVMware上CentOS6に環境を構築しているので、どこでも編集することができない
    → どこの環境でも手軽に更新できるようにしたい
    → EC2に環境を構築すればそこにアクセスすればiPad, iPodtuchでも編集できる。(iOSのAWSアプリを使えばモバイルから手軽にインスタンスの起動と停止できるよね)
    → 作業フォルダをDropboxに置けば、記事は常に最新に保てる
    → EC2にDropboxって設定できるのかな
    → できた

  • ブログの画像ファイル置き場作成
    → Dropboxのpublicフォルダを使えばいいかなぁ
    → できた
